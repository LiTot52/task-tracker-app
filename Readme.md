
# Task Tracker App

Веб-приложение для управления задачами. Курсовой проект, СКФУ 2026.

## Технологии

- HTML5
- CSS3 (Flexbox, Grid, CSS-переменные)
- JavaScript ES6+
- localStorage

## Функциональность

- Добавление, редактирование и удаление задач
- Установка приоритета и дедлайна
- Фильтрация по статусу (все / активные / выполненные)
- Поиск задач в реальном времени
- Сохранение данных в localStorage
- Тёмная и светлая тема

## Запуск

Открыть `index.html` в браузере — установка не требуется.
